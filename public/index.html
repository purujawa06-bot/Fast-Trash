Selamat datang di dokumentasi API kami yang sangat ramah. Di sini, kamu akan menemukan semua yang perlu kamu ketahui untuk bermain dengan endpoint kami yang luar biasa!

GET /

Ini adalah markas utama! Kunjungi endpoint ini untuk mendapatkan sambutan hangat dari server kami. Sempurna untuk memeriksa apakah semuanya berjalan lancar.

Coba ini!

curl -X GET /

Atau cukup buka browser-mu ke /

Respon yang Diharapkan:

{"message": "Selamat datang di Simple API!"}

GET /api/ping

Ingin tahu apakah server kami terjaga dan siap berpesta? Kirim "ping" dan ia akan dengan senang hati mengirim "pong" kembali!

Coba ini!

curl -X GET /api/ping

Atau cukup kunjungi /api/ping di browser-mu.

Respon yang Diharapkan:

{"message": "pong"}
